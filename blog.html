Important FAQ

What is the Document Object Model (DOM) ?

The DOM, or Document Object Model, represents the structure of web pages, enabling dynamic manipulation of content and styles.

How do you select an element from the DOM ?

Use document.querySelector() or document.getElementById() to select DOM elements easily.

What is event delegation in the context of the DOM, and why is it useful

Event delegation is a technique in the DOM where a single event listener is added to a parent element instead of individual listeners on child elements. When an event occurs on a child, it bubbles up to the parent, triggering the listener.

How do you manipulate an element's attribute and styles using the DOM

You can manipulate attributes with getAttribute(), setAttribute(), and removeAttribute(). For styles, use element.style or classList.
